# Animation and visual system

## Creative direction

Everything Media should feel like an AI studio directed by filmmakers, not a generic AI agency. The design language uses cinema references: frame lines, lens metadata, black rooms, copper grease-pencil accents, glass depth, hard cuts, slow parallax, and camera-like motion.

The UI should feel premium because it is restrained. Every animation has to feel intentional, not decorative.

## Visual language

### Color

- Base: near-black, not pure black.
- Type: warm bone white.
- Accent: copper red, used sparingly.
- Secondary accent: muted gold for highlights and gradients.
- Surfaces: translucent glass layers with thin borders.

### Typography

- Display: Instrument Serif, large, editorial, high contrast.
- UI: JetBrains Mono, small, technical, cinematic labels.
- Body: Inter, clean and modern.

The display type should do most of the emotional work. Avoid too many colors or too many font weights.

### Layout

- Large empty areas should feel like darkness around a screen.
- Headlines are oversized and slightly poetic.
- Details are small and precise: scene numbers, lens notes, timecodes, labels.
- Work should feel like a reel, not a grid.

## Animation principles

### 1. Camera, not website

Motion should feel like camera movement: drift, parallax, focus, hard cuts, light leaks, and slow reveal.

### 2. Slow premium entry

The hero does not rush. It loads with a calibration frame, then the title rises like a reveal in a trailer.

### 3. Small UI details move fast

Buttons, cursor, timecodes, and hover states are quick and tactile.

### 4. Big objects move slowly

The lens, glass plates, ambient orbs, and background particles move slowly to create depth.

### 5. Scroll is editorial

The horizontal reel turns scrolling into a film strip. Each card feels like a frame from a larger story.

## Included animation components

### Preloader

A calibration frame with count-up metadata. It sets the cinematic tone immediately.

### Hero stills

The hero image hard-cuts between cinematic stills with a short flash. This references film editing rather than a soft slideshow.

### CSS 3D lens

The lens object is made with CSS rings, transforms, and gradients. It rotates and floats subtly, then reacts to mouse movement.

### Glass plates

Floating glass UI elements introduce production language like "prompt", "taste check", "frame", and "locked".

### Particle canvas

A lightweight 2D canvas creates an AI constellation field. It should stay subtle and never compete with the copy.

### Custom cursor

The cursor is a viewfinder. It expands on interactive elements and reinforces the studio/filmmaker feeling.

### Bento card tilt

Capability cards respond to pointer movement with subtle 3D tilt and a copper light source.

### Horizontal reel

On desktop, the work section is sticky and converts vertical scroll into horizontal motion. On mobile, it becomes a native swipe carousel.

### Lab object

The lab section uses layered frames and orbit rings as a visual metaphor for worldbuilding, consistency, and shot grammar.

## Copy direction

Original core kept:

- "An AI film studio, built by filmmakers."
- "Tools change. Taste doesn't."

Improved positioning:

- "The tool can make frames. Taste decides which frames matter."
- "AI is not the director. It is the camera, the crew, the set, and the first draft. Direction is still the work."
- "Let's make something that feels expensive before the logo appears."

2. Add real case study pages for each work card.
3. Add a short background showreel video in the hero after you have final material.
4. Add a real form backend.
5. Replace Google fonts with licensed premium fonts if the brand budget allows it.
